# Spiderman-Inhome

SpiderVision is a computer vision project inspired by Spider-Man.

The application uses a laptop webcam to detect a Spider-Man hand gesture in real time. When the gesture is detected, a web animation appears on the screen. Pulling the hand backward triggers a smooth camera zoom effect, creating an interactive AR-like experience.

## Features

* Real-time hand tracking
* Spider-Man gesture detection
* Animated web effect
* Motion-based camera zoom
* Smooth visual transitions
* Webcam-based interaction

## Technologies Used

* Python
* OpenCV
* MediaPipe
* NumPy
